🎯 系統概述

這是一個專為小型單位設計的請假管理系統，使用 Google Sheets 作為資料庫，提供完整的請假申請、審核和管理功能。

開始體驗 員工介面 管理員介面

✨ 主要功能

  • 員工和管理員雙角色登入系統
  • 個人假期餘額即時查看
  • 完整的請假申請流程
  • 管理員審核和核准功能
  • 請假紀錄查詢和管理
  • 員工資料管理
  • 系統統計和報表
  • 響應式設計，支援行動裝置

💻 技術架構

前端技術

HTML5, CSS3, JavaScript

後端技術

Google Apps Script

資料庫

Google Sheets

部署平台

GitHub Pages / Netlify

📊 資料庫設計

資料表 主要欄位 用途
員工基本資料 員工ID、姓名、Email、到職日、假期餘額 儲存員工基本資訊和假期資料
請假紀錄 申請ID、員工ID、請假日期、假別、狀態 記錄所有請假申請和審核狀態
假期類型 假別代碼、假別名稱、是否扣薪 定義系統支援的假期類型

🔄 使用流程

員工流程

  1. 登入系統
  2. 查看假期餘額
  3. 填寫請假申請
  4. 提交申請
  5. 查看審核結果

管理員流程

  1. 登入管理員介面
  2. 查看待審核申請
  3. 審核請假申請
  4. 核准或拒絕
  5. 管理員工資料
